# Page Structure

## Public Pages

/

Landing page.

Displays:

- hero section
- couple story preview
- link to timeline

---

/wedding

Wedding portal.

Sections:

- couple hero
- wedding date
- timeline preview
- guest wishes
- upload photo

---

/timeline

Relationship timeline.

Displays chronological events.

---

/event/[id]

Single event page.

Displays:

- event details
- associated media
- comments or notes

---

/gallery

Media gallery.

Grid layout.

---

/letters

Letters archive.

List letters.

---

/profile

Couple profile.

Displays:

- avatars
- relationship start
- story

---
