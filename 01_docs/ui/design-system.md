# Design System

## Design Philosophy

Project 2035 is a romantic memory archive.

The UI must feel:

- calm
- elegant
- minimal
- emotional

Avoid clutter.

---

## Color Palette

Primary:

rose / pink gradient

Example:

bg-gradient-to-r from-rose-400 to-pink-500

Secondary:

soft neutral backgrounds

Example:

bg-zinc-50
bg-white

Accent:

gold highlight for special events.

---

## Typography

Heading:

serif style for emotional tone.

Example:

font-serif text-3xl font-semibold

Body text:

clean sans-serif.

Example:

text-sm text-muted-foreground

---

## Spacing

Use consistent spacing scale.

Common:

p-4
p-6
p-8

Sections:

py-16

---

## Component Patterns

Cards

Used for:

events
letters
media

Dialogs

Used for:

media preview
confirm actions

Grid

Used for:

gallery

---

## Icons

Use Lucide icons.

Heart
