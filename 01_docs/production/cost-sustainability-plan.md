# Cost Sustainability Plan

## 1. Purpose

Ensure project remains financially sustainable long-term.

---

## 2. Estimated Monthly Cost

Vercel (Hobby or Pro)  
PostgreSQL (Supabase / Neon)  
Storage (S3 / Cloudinary)  
Domain renewal  

Total estimated:
$10–30/month (basic usage)

---

## 3. Long-Term Rule

If usage decreases:

- Downgrade plan
- Disable unnecessary features
- Switch to lower-cost DB tier

---

## 4. Budget Threshold

If monthly cost exceeds defined threshold:

- Review storage usage
- Review media size
- Consider archive compression

---

## 5. Annual Review

Once per year:

- Review hosting plan
- Review database usage
- Review storage growth
- Optimize media if necessary

---

## 6. Sustainability Principle

The system must be affordable even if income changes.

Keep it simple.
